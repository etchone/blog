我的第一篇博客 & 搭建指南

# 前言

说来很久之前就已经有零零散散地搭过几次的博客，但当时的我都是以研究为目的去搭它们的 (难怪那些博客存活时间都不长)

不过这一个应该会作为我今后日常使用的博客了 (所以这个算是正式版了？应该不会 404 吧...)

这篇文章就简要介绍一下这个博客吧

# 博客架构简介及搭建教程

  1. 本地: Hexo

    快速、简洁且高效的博客框架

    配置完它之后，我们在本地就有地方写博客啦

    官网地址: https://hexo.io/

    教程见下方

  2. 服务器: Github

    世界上最大的同性交友网站 面向开源及私有软件项目的托管平台

    通过设置 Github pages, 我们可以拥有一个形如 NAME.github.io 的域名

    官网地址: https://github.com/

    Github 搭建个人博客 Github page 教程: 使用 hexo+github 搭建免费个人博客详细教程 - 小茗同学

  3. 域名提供：阿里云 (此步付费！！)

    上云就上阿里云

    官网地址: https://www.aliyun.com/

    注册教程: 如何在阿里云申请域名 - 百度经验

  4. 域名解析与 HTTPS 提供: Cloudflare

    保护并加速您的网站，应用和团队

    中文官网: https://www.cloudflare.com/zh-cn/network/china/

    域名设置 Cloudflare 解析教程: 如何给你的网站套上 Cloudflare（以阿里云为例） - Bottle

    解析 & HTTPS 教程: Github pages 以个人域名实现 https 访问 - Judikator
